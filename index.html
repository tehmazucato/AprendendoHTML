Como surgiu a Internet?

Clique aqui para voltar

A internet é uma rede global de computadores interconectados que permite a troca de informações e comunicação entre pessoas em todo o mundo.

Sua história remonta ao final dos anos 1960, quando o Departamento de Defesa dos Estados Unidos desenvolveu a ARPANET, uma rede de comunicação experimental que permitia que computadores trocassem informações entre si.

A ARPANET foi o precursor da internet moderna e começou a ser usada em 1969.

Nos anos subsequentes, várias tecnologias e protocolos foram desenvolvidos para permitir a comunicação eficiente entre computadores.

Um marco importante foi a criação do TCP/IP, chamado de Transmission Control Protocol ou Internet Protocol que estabeleceu um conjunto de regras para a transmissão de dados através de redes de computadores, permitindo que redes diferentes se conectassem e trocassem informações.

Nos anos 1980 e 1990, a World Wide Web, conhecido como WWW, foi desenvolvida como uma forma de organizar e acessar informações na internet.

A WWW introduziu a ideia de páginas da web interconectadas através de hiperlinks, permitindo que os usuários navegassem de uma página para outra.

Isso levou à popularização da internet como a conhecemos hoje.

O que é HTML?

Clique aqui para voltar

HTML é uma linguagem de marcação utilizada para criar documentos, quais sejam páginas da web que são exibidos em navegadores.

Ele utiliza uma estrutura baseada em "tags" para definir a estrutura e o conteúdo de uma página da web.

Cada tag HTML possui uma funcionalidade específica.

Como funcionam as TAGS?

Clique aqui para voltar

As tags são elementos fundamentais da linguagem HTML e são usadas para estruturar e formatar o conteúdo de uma página da web.

Cada tag é cercada por colchetes angulares ("<" e ">") e pode ter atributos que fornecem informações adicionais sobre a tag.

As tags HTML são interpretadas pelos navegadores da web, que renderizam o conteúdo de acordo com as instruções fornecidas pelas tags.

  1. Identificação e Abertura de Tags:

    2. Cada tag é reconhecida pelo navegador a partir dos colchetes angulares ("<" e ">"). A tag de abertura indica o início de um elemento.

    Por exemplo, a informação < h1 > é uma tag de abertura para um título de nível 1.

  2. Conteúdo da Tag:

    3. Algumas tags contêm conteúdo visível na página, como texto, imagens ou outros elementos. O conteúdo é inserido entre a tag de abertura e a tag de fechamento correspondente.

    Por exemplo: < p >Isso é um parágrafo de texto.< /p >

  3. Tags de Fechamento:

    4. A maioria das tags HTML precisa de uma tag de fechamento correspondente para indicar onde o elemento termina. As tags de fechamento têm uma barra ("/") antes do nome da tag.

    Por exemplo, < /p > é uma tag de fechamento para o parágrafo acima.

  4. Tags Auto-fechadas:

    5. Algumas tags não têm conteúdo associado e, portanto, não precisam de uma tag de fechamento separada. São chamadas de tags auto-fechadas.

    Como por exemplo: < img src="imagem.jpg" alt="Imagem" >

  5. Atributos:

    6. As tags podem ter atributos que fornecem informações adicionais sobre o elemento. Os atributos são definidos dentro da tag de abertura e podem conter valores que afetam o comportamento ou a aparência do elemento.

    Por exemplo: < a href="https://www.exemplo.com.br" > Clique aqui< /a >

  6. Hierarquia e Estruturação:

    7. As tags HTML permitem criar a estrutura básica de uma página da web, com cabeçalhos, parágrafos, listas, imagens, links e mais. Ao combinar diferentes tags e elementos, você pode criar layouts e estilos para tornar a página visualmente atraente e funcional.

    Entre muitos outros exemplos de TAGS que podem e devem ser utilizadas quando utiliza-se o HTML.

    Cabe destacar que as tags HTML são a base da criação de páginas da web. Elas definem a estrutura e o conteúdo da página, permitindo que os navegadores interpretem e exibam o conteúdo de acordo com as instruções fornecidas pelas tags.
